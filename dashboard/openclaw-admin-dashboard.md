# OpenClaw Local Admin Dashboard

## Features
- Real-time view of all Akash leases and mining activity
- Shard harvesting cron status
- Marketing campaign performance
- Immunefi bug bounty dashboard
- Wallet connection status for all equipment
- Runic Language execution logs
- GEOD crons and geospatial data
- Odysseus ChromaDB memory: mutations, performance, Deity identities, learnings
- Cross-node memory sync status for Akash workers and multi-cloud nodes

## Local Instance
Run locally on user hardware or via cloud credits. Accessible via browser or Telegram.

## Connected Equipment
- Bminer instances
- lolMiner instances
- SRBMiner-MULTI
- Current coins being mined (to be detected via pool/API)
- Payout wallet configuration

## Odysseus Memory Panel
- Query shared long-term memory across all 10,080 mutated agents
- Inspect mutation and performance history by `agent_id` or `AGENT_SHARD_ID`
- Confirm 169 Deity identity records and 14-Council authority metadata
- Monitor `/odysseus/memory/sync` peer health, cursors, imports, and pushes
